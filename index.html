Intro

Reeba Qureshi

I am a Computer Science undergraduate, currently in second year, pursuing my BTech from Bharati Vidyapeeth's College of Engineering. A born observer and geek, I was fascinated with technology since the beginning. When I studied Python as a part of my school's curriculum, I realized this is what I wanted to pursue as a career. After exploring many fields, I chose Machine Learning; I was enthralled by the idea of being able to predict the future using past data!

I studied Data Science and Machine Learning independently, from scratch, utilizing the limitless resources on the internet. I culminated knowledge and worked on projects which ultimately led me here!

Aside from the technical domain, I'm a huge bookworm, with a special love for psychological thrillers. I paint and sing as a form of expression and enjoy playing musical intruments like the piano, guitar and ukulele.

I built this website to maintain a track of my technical journey and showcase my projects.

Projects

DocHome

Multi-Layer Perceptrons, Natural Language Processing, Web Scraping

A web based application that can take user entered symptoms and predict the diseases the user might be suffering from along with percentage accuracy and corresponding remedies/ suggestions.

Newgroups Text Classification

Multinomial Naive Bayes, Natural Language Processing

A text classifier that analyzes the text of an article and classifies it into one of 20 newgroups using Multinomial Naive Bayes from sklearn library as well as by self implementation.

Fashion MNIST Classification

Neural Network, TensorFlow, Keras

Created an image classifier for Fashion-MNIST, consisting of a training set of 60,000 examples and a test set of 10,000 examples. using Sequential Model in Keras with 88.18% accuracy.

CIFAR-10 Image Classification

Sequential Models, TensorFlow, Keras

Built an image classifier for CIFAR-10 dataset containing 60000 32x32 colour images in 10 classes, with an accuracy of 72%.
